How do I calculate the integer number of vertices in a 2 dimensional triangle?

Created April 5, 2020
  • Single Issue
  • College level
  • Math
Imagine a 2-dimensional right-angled triangle drawn on graph paper (i.e. a lattice), with the right-angle corner originating at (0,0). Each unit the graph paper has a width of 1 unit. The lengths of the base and height for this triangle can be any real number. Is there a formula for determining the number of lattice points contained in the triangle?
